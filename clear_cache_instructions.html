🚨 חובה לנקות Cache!

⚠️ הבעיה: הדפדפן שלך משתמש בקבצי JavaScript ישנים!

🧹 שלב 1: ניקוי Cache מלא

בחר אחת מהאפשרויות:

אופציה א׳ - מהיר (Chrome/Edge):

  1. פתח את האתר
  2. לחץ Ctrl + Shift + F5
  3. או לחץ F12 → לחצן ימני על כפתור הרענון → Empty Cache and Hard Reload

אופציה ב׳ - מלא:

  1. לחץ Ctrl + Shift + Delete
  2. בחר Cached images and files
  3. בחר All time
  4. לחץ Clear data

אופציה ג׳ - גלישה בסתר:

  1. פתח חלון גלישה בסתר (Ctrl + Shift + N)
  2. היכנס לאתר משם

📝 שלב 2: עדכון קבצי JavaScript

צריך לשנות את מספרי הגרסה בקבצים כדי לכפות טעינה מחדש:

# רק אם אתה רוצה לכפות עדכון לכל המשתמשים: # פתח את index.html או company-manager.html # חפש את השורות עם: parking-api-live.js?v=1734048960 parking-ui-live.js?v=1734048962 # שנה ל: parking-api-live.js?v= # או פשוט מספר חדש: parking-api-live.js?v=2024121501 parking-ui-live.js?v=2024121501

✅ שלב 3: בדיקה

אחרי הניקוי, בדוק שוב:

פתח את האתר בדוק Proxy
💡 טיפ: אם עדיין לא עובד אחרי ניקוי Cache, נסה:
  1. התנתק מהמערכת והתחבר מחדש
  2. השתמש בדפדפן אחר
  3. בדוק בטלפון

🔍 איך לבדוק שהניקוי עבד?

פתח F12 → Network → רענן את הדף

בדוק שהקבצים נטענים בלי (from cache)
