===== הוראות לעדכון קובץ .env =====

פתח את הקובץ .env בעורך טקסט ועדכן:

1. שנה את השורה:
   מ: SUPABASE_URL=https://your-project-id.supabase.co
   ל: SUPABASE_URL=https://[הכתובת-האמיתית-שלך].supabase.co

2. הוסף שורה חדשה אחרי SUPABASE_URL:
   SUPABASE_ANON_KEY=[המפתח-האמיתי-שלך]

3. אם אתה רוצה להשתמש ב-Gmail במקום 012.net.il, שנה את:
   EMAIL_USER=Report@sbparking.co.il
   EMAIL_PASS=o51W38D5
   
   ל:
   GMAIL_USERNAME=your-email@gmail.com
   GMAIL_APP_PASSWORD=your-app-password

4. שמור את הקובץ

5. הפעל מחדש את האפליקציה:
   python appV3.py

===== איפה למצוא את הפרטים ב-Supabase =====

1. היכנס ל: https://app.supabase.com
2. בחר את הפרויקט שלך
3. לך ל: Settings (הגדרות) → API
4. תמצא שם:
   - Project URL (זה ה-SUPABASE_URL)
   - anon public (זה ה-SUPABASE_ANON_KEY)

===== בדיקה שהכל עובד =====

אחרי שתעדכן ותפעיל מחדש, אמור להופיע:
✅ Supabase URL: ✅ SET
✅ Supabase KEY: ✅ SET
✅ Supabase connection established
